# Proyecto desplegado
El proyecto Parisi Motors se encuentra desplegado en producción, a continuación adjunto los links:

Para la Aplicación web (Frontend) se ha utilizado vercel: https://concesionario-front-navy.vercel.app/
Para la Api (Backend) y la base de datos se ha utilizado Railway: https://concesionario-back-production.up.railway.app

# Para realizar el despliegue local
**Primero se debe inicializar el backend**
Documentación para el despliegue local del backend: https://github.com/Ramiroparisi/Concesionario-Back/blob/main/README.md

## Frontend
Link al repositorio: https://github.com/Ramiroparisi/concesionario-front

### Requisitos
Se debe contar con [Node.js](https://nodejs.org/) (v18 o superior)

### Ejecución

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
### Entorno
Una vez incializado, el entorno del frontend se encuentra en [http://localhost:3001](http://localhost:3001)
